# Proyectos Substack - Manuel A. Hidalgo Pérez

Bienvenido al directorio de proyectos y publicaciones para [Substack](https://substack.com/@manujhidalgo).

## Descripción

Este espacio está dedicado a la creación, análisis y preparación de contenidos para la newsletter y perfil de Substack de **Manuel A. Hidalgo Pérez**, profesor de economía y divulgador. El objetivo principal de estos artículos es explicar la actualidad económica de forma clara y accesible.

Aquí se alojan:
*   **Documentos (md, docx, pdf)**: Borradores, revisiones de literatura y artículos finales.
*   **Scripts (scr)**: Códigos y análisis de datos (Python, R, etc.) utilizados para generar gráficos y extraer conclusiones económicas.
*   **Datos (data)**: Conjuntos de datos y archivos CSV utilizados como fuente de información.
*   **Imágenes**: Gráficos y recursos visuales para acompañar los posts.

## Estructura de Proyectos

Cada carpeta principal dentro de este directorio corresponde generalmente a un tema, artículo o análisis específico (por ejemplo: `Descomposición salarios reales`, `horas epa`, etc.).
